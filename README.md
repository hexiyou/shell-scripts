# shell-scripts

#### 介绍
一些实用的Linux Bash/Shell脚本收藏集，Windows系统下Cygwin、WSL，Git for Windows Bash、Msys2等模拟环境中亦可使用;

建立此仓库目的主要为了方便统一收集和管理不同功能的Shell脚本，同时对shell脚本代码增加版本控制，方便回溯；
